### the C++ version of centernet_mobilenetv2 with ncnn 

## requirement
- ncnn
- opencv(optional)

## usage 
 * cd centernet_mobilenetv2_ncnn
 * mkdir build && cd build && cmake && make
 * ./demo ../models/  your_image_path
